==Description==

Flexi Related Posts is a WordPress related posts content plugin and widget with responsive layout for mobile devices. Flexi Related Posts displays your blog's related posts using beautiful widget. 

Flexi Related Posts is light weight touch supported, responsive posts content widgets and plugin. Manage your posts to show in sidebar With pre-designed layouts to save your time. Simple and easy to use, no coding skills required.

Features:

* Responsive related post widget.
* Fully Customizable.
* Select how many posts to display and also select date range to show.
* Select Vertical or Horizontal Layout.
* customizable style sheet.
* Select to show with content or without content.
* Fast support + Free Updates.

== Frequently Asked Questions ==

= How can I use this plugin as a widget =

When you install this plugin, it will activate the Flexi Related Post Widget. You can go in Appearance -> Widget area and add the widget in desired side bar.

 == Installation ==
* Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
* Find menu item entitled Flexi Related Posts.
* Create Your Default Settings for Display Related Posts with Single Posts of the site.
